◆ validate() [1/3]

ArrayDetails chihaya::validate ( const H5::Group &  handle)
inline

Validate a delayed operation/array at the specified HDF5 group, using a version string in the delayed_version attribute of the handle. This should be a string of the form <MAJOR>.<MINOR>. For back-compatibility purposes, the string "1.0.0" is also allowed, corresponding to version 1.1; and if delayed_version is missing, it defaults to 0.99.

Parameters
handleOpen handle to a HDF5 group corresponding to a delayed operation or array.
Returns
Details of the array after all delayed operations in handle (and its children) have been applied.
