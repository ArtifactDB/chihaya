Detailed Description

Validation options.

This is used to configure options for a single call to validate(). It can be used to override validation functions without modifying the global validation registries, e.g., for parallelized applications. The options may also mutate throughout the duration of the call, allowing callers to collect statistics across the recursive invocations of validate() on the same Options object.
